# 小鸭子冒泡排序可视化动画项目

## 项目介绍

小鸭子冒泡排序可视化动画是一个使用Python + Tkinter开发的交互式教学工具，通过可爱的鸭子图形动画直观展示冒泡排序算法的工作原理。该项目包含12只不同大小的小鸭子和1只大母鸭，通过动画演示排序过程中元素的比较和交换操作。

这是一个使用Python + Tkinter开发的冒泡排序可视化动画教学工具。通过12只不同大小的小鸭子和一只大母鸭的动画演示，帮助用户直观理解冒泡排序算法的工作原理。

## 使用说明

### 基本操作

1. **启动程序**：运行程序后会看到主界面，包含12只随机大小的小鸭子和1只大母鸭
2. **开始排序**：点击"开始排序"按钮，开始冒泡排序动画演示
3. **控制动画**：
   - **暂停**：点击"暂停"按钮暂停动画
   - **继续**：点击"继续"按钮恢复动画播放
   - **重置**：点击"重置"按钮重置到初始状态
   - **单步执行**：点击"单步执行"按钮逐步执行排序
4. **生成新数据**：点击"新的小鸭子"按钮生成新的随机数据

### 速度控制

- 使用速度控制滑块调节动画速度（0.1x - 3.0x）
- 速度值会实时显示在滑块下方

### 统计信息

程序会实时显示以下统计信息：
- **比较次数**：当前已进行的元素比较次数
- **交换次数**：当前已进行的元素交换次数
- **进度**：排序完成的百分比
- **进度条**：可视化显示排序进度

### 状态信息

程序会显示以下状态信息：
- **状态**：程序当前状态（就绪、排序中、已暂停、排序完成）
- **当前操作**：当前正在执行的操作（比较或交换的具体位置）
- **排序状态**：排序算法的状态
- **动画状态**：动画系统的状态

## 鸭子设计说明

### 小鸭子特征

- **大小变化**：根据数值大小在20-50像素范围内变化，数值越大鸭子越大
- **颜色方案**：
  - 默认：金黄色身体
  - 高亮：粉红色
  - 比较中：深青色
  - 已排序：绿色
- **组成元素**：椭圆形身体、圆形头部、三角形嘴巴、小眼睛、翅膀
- **数值显示**：每只鸭子下方显示其代表的数值

### 大母鸭特征

- **大小**：固定60像素，比小鸭子大1.5倍
- **颜色方案**：棕色系身体，带有金色皇冠
- **特殊标识**：皇冠表示身份，"母鸭"文字标签
- **动画效果**：会移动到比较位置、指向比较的鸭子、点头表示理解

## 排序算法说明

本程序使用经典的冒泡排序算法：

1. **基本原理**：重复遍历要排序的数列，一次比较两个元素，如果顺序错误就交换
2. **算法特点**：
   - 时间复杂度：O(n²)
   - 空间复杂度：O(1)
   - 稳定排序
3. **可视化过程**：
   - 大母鸭移动到需要比较的鸭子附近
   - 高亮显示正在比较的两只鸭子
   - 如果需要交换，两只鸭子会以弧形路径交换位置
   - 已排序的鸭子会变成绿色

## 技术架构

### 核心模块

1. **图形系统**（graphics.py）
   - `Duck`：鸭子基类，定义基本属性和行为
   - `BabyDuck`：小鸭子类，支持不同大小和状态显示
   - `MotherDuck`：大母鸭类，带有特殊动画效果
   - `DuckFactory`：鸭子工厂类，批量创建鸭子对象

2. **排序算法**（algorithms/bubble_sort.py）
   - `BubbleSort`：冒泡排序算法实现，支持逐步执行和状态跟踪

3. **动画系统**（animation/）
   - `AnimationEngine`：动画引擎，管理所有动画效果
   - `Animators`：各种专门的动画器类
   - `SortAnimationIntegration`：排序算法与动画系统的集成

4. **主程序**（main.py）
   - `DuckBubbleSortApp`：主应用程序类，管理GUI和用户交互

### 设计模式

- **工厂模式**：`DuckFactory`用于创建鸭子对象
- **观察者模式**：排序算法通过回调函数通知动画系统
- **策略模式**：不同的动画器实现不同的动画效果
- **单例模式**：动画引擎确保全局唯一

## 故障排除

### 常见问题

1. **tkinter未安装**
   - **Ubuntu/Debian**：`sudo apt-get install python3-tk`
   - **Fedora**：`sudo dnf install python3-tkinter`
   - **macOS**：通常已包含，如有问题请重新安装Python
   - **Windows**：通常已包含，如有问题请重新安装Python

2. **程序启动失败**
   - 检查Python版本是否为3.6+
   - 确保所有项目文件完整
   - 使用`python run.py`启动以获取详细错误信息

3. **动画卡顿**
   - 降低动画速度
   - 关闭其他占用CPU的程序
   - 检查系统资源使用情况

4. **窗口显示异常**
   - 确保系统支持GUI显示
   - 检查屏幕分辨率是否足够（最低要求1200x800）
